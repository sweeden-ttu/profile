# CLAUDE.md

## Project Overview
Jekyll-based blog and portfolio for **Scott Weeden**, focusing on CS research and Logic Programming.

## Simplified Mission
Develop and maintain a **Herbrand Base Prolog Application** for the CS-5384 Logic for Computer Scientists course.

## Core Rules
1.  **Prolog First**: All logical reasoning components should be implemented in `scripts/herbrand.pl`.
2.  **Instructional Content**: Blog posts in `_posts/` should document the Herbrand base logic.
3.  **Math Standard**: Use KaTeX for all formulas (`$...$` inline, `$$...$$` display).
4.  **Consistency**: Refer to `AGENT_PROMPTS.md` for simplified agent tasks.

## Key Files
- `scripts/herbrand.pl`: The core logic application.
- `AGENTS.md`: System overview.
- `AGENT_PROMPTS.md`: Task instructions.
- `math-rules.md`: Mathematical notation standards.
