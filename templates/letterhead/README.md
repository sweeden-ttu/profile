# Cloud Fronts Group — Texas 2025 Letterhead

Professional letterhead for the franchise public report submitted to the Texas Comptroller.

## Files

- **cloud-fronts-group-texas-2025-letterhead.html** — Full-page letterhead. Open in a browser and use Print → Save as PDF for a standalone first page or letterhead PDF.

## Use

1. Open the HTML file in a browser.
2. Print to PDF (Ctrl+P / Cmd+P → Save as PDF) for a letter-size letterhead page.
3. Use the generated PDF as the first page of the Texas 2025 Report of Profit & Loss, or as a template for official filings.

- **Company:** Cloud Fronts Group
- **Document:** Texas 2025 Report of Profit & Loss
- **Recipient:** Office of the Texas Comptroller of Public Accounts (Comptroller's Office of Currency / franchise tax reporting)
